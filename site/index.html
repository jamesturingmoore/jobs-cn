中国职业市场可视化 GitHub

基于《中华人民共和国职业分类大典》(2022年版)的职业数据可视化工具。每个矩形的面积与就业人数成正比。颜色显示选定的指标——可在增长前景、薪资水平、学历要求和AI暴露度之间切换。点击任意方块可查看详细信息。这是一个开发工具，用于可视化探索中国职业市场数据。

LLM驱动的评分: 本项目包含数据采集、解析和LLM评分管道。您可以编写自定义提示词，让LLM对每个职业进行评分，树状图将相应着色。"AI暴露度"选项是一个示例——它估计当前AI（主要是数字化AI）将如何重塑每个职业。您也可以编写不同的提示词来评估其他维度——例如人形机器人风险、离岸外包风险、气候影响等。

你是一位专业的劳动力市场分析师。阅读以下来自中国招聘平台的职业描述。 请评估该职业的整体 **AI暴露度**，评分范围 0-10。 AI暴露度衡量：AI将如何重塑这个职业？请同时考虑： - 直接影响：AI自动化目前由人类完成的任务 - 间接影响：AI使每位工作者效率提升，导致需求减少 关键信号：该职业的工作产品是否从根本上属于数字化。 如果工作可以在家庭办公室完全通过计算机完成，AI暴露度天生就很高（7+）。 评分锚点： - 0-1：最低暴露（建筑工人、园林工人） - 2-3：低暴露（电工、水管工、消防员） - 4-5：中等暴露（注册护士、警察、兽医） - 6-7：高暴露（教师、经理、会计师、记者） - 8-9：极高暴露（软件开发、平面设计、翻译、数据分析师） - 10：最大暴露（数据录入员、电话推销员） 请仅以JSON格式回复： {"exposure": <0-10>, "rationale": "<2-3句解释关键因素>"}

关于AI暴露度的说明: 这些是LLM的粗略估计，不是严谨的预测。高分并不意味着该职业会消失。软件开发人员的得分为9/10，因为AI正在改变他们的工作——但随着每位开发者效率的提升，对软件的需求可能会增长。评分不考虑需求弹性、潜在需求、监管障碍或对人类工作者的社会偏好。许多高暴露度职业将被重塑，而非被取代。
